## Glosario rápido

- **MVP**: *Minimum Viable Product* — producto mínimo viable para aprender.
- **MMP / MMR / MMF**: variantes mínimas para el mercado / release / feature.
- **PBI**: *Product Backlog Item*.
- **DoR / DoD**: *Definition of Ready / Done*.
- **SM / PO / DT**: Scrum Master, Product Owner, Development Team.
- **WIP**: *Work In Progress* — trabajo en curso.
- **RTF**: *Running Tested Features*.
- **PT**: *Product Backlog*.
- **SBT**: *Sprint Backlog*.
- **NIT**: *Nexus Integration Team*.
