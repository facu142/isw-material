## Glosario rápido

- **SCM**: *Software Configuration Management* — gestión de configuración de software.
- **SCI / IC**: *Software Configuration Item* — ítem de configuración.
- **PCA**: *Physical Configuration Audit* — auditoría física de configuración.
- **FCA**: *Functional Configuration Audit* — auditoría funcional de configuración.
- **CI**: *Continuous Integration* — integración continua.
- **CD**: *Continuous Delivery* (entrega continua) / *Continuous Deployment* (despliegue continuo).
- **Blue-Green**: estrategia de despliegue con dos entornos idénticos.
- **Canary**: estrategia de despliegue gradual a un % de usuarios.
